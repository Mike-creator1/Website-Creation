# Website Creation 🌐

Demo websites για τοπικές επιχειρήσεις στην Ελβετία.
Βρίσκουμε επιχειρήσεις στο Google Maps → φτιάχνουμε demo → τους στέλνουμε ζωντανό link.

## 📁 Δομή

```
Website-Creation/
├── _TEMPLATE/          ← Το βασικό template. ΜΗΝ το πειράζεις — το ΑΝΤΙΓΡΑΦΕΙΣ.
│   └── index.html
├── demos/              ← Εδώ μπαίνει ΕΝΑΣ φάκελος για κάθε επιχείρηση
│   └── beispiel-restaurant-zuerich/
│       └── index.html
└── README.md
```

## 🚀 Πώς φτιάχνω ένα νέο demo

1. Αντιγράφω τον φάκελο `_TEMPLATE` μέσα στο `demos/` και του δίνω όνομα με την επιχείρηση.
   Παράδειγμα: `demos/friseur-mueller-bern/`
2. Ανοίγω το `index.html` μέσα σε αυτόν τον φάκελο.
3. Αλλάζω μόνο ό,τι λέει `👉 ΑΛΛΑΞΕ` (όνομα, τηλέφωνο, διεύθυνση, κείμενα, χρώματα, φωτογραφίες).
4. Το ανοίγω με διπλό κλικ για να το δω στον browser.
5. Push στο GitHub και deploy για να πάρω link που στέλνω στην επιχείρηση.

## 👥 Συνεργασία (εγώ + αδερφός)

**Χρυσός κανόνας:** `git pull` πριν αρχίζεις, `git push` όταν τελειώνεις.

Ο καθένας δουλεύει σε **δικό του φάκελο** μέσα στο `demos/` → ποτέ conflict.

```
git pull
# ... φτιάχνω το demo μου ...
git add .
git commit -m "Demo για [όνομα επιχείρησης]"
git push
```
